React.js
Node.js
Express.js
MySQL
Recharts
JWT Authentication

✅ User Authentication (Login & Signup)

✅ Positive / Negative / Neutral Analysis
✅ Bar Graphs
✅ CSV Upload & Bulk Prediction
✅ Export PDF Reports

✅ MySQL Database Integration
